# 🍽 Hostel Food Wastage Reduction System

A web-based system designed to monitor, analyze, and reduce food wastage in hostel mess facilities through structured data tracking and analytics.

---

## 🚀 Project Overview

The Hostel Food Wastage Reduction System helps hostel management:

- Track daily food wastage
- Analyze waste patterns
- Monitor trends over time
- Make data-driven decisions to reduce food loss

The system provides structured database integration and an interactive interface for better operational efficiency.

---

## 🎯 Key Features

- Daily food waste entry and monitoring
- Trend analysis and visualization
- Structured database integration using Supabase
- Environment-based configuration
- Fast frontend powered by Vite and Tailwind CSS

## ⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/drashtisojitra2411-jpg/Hostel-Food-Wasatage-Reduction.git

2. Navigate to the project folder

cd Hostel-Food-Wasatage-Reduction

3. Install dependencies

npm install

4. Configure Environment Variables

Create a file named .env.local and add:

SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key

5. Run the project

npm run dev
